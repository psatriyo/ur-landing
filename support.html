Frequently Asked Questions

‚ùì How do I get started with UangRumah?

Getting started is easy:

  1. Download UangRumah from the App Store
  2. Set up your monthly and weekly budgets in Settings
  3. Create custom categories and cost centers for your spending
  4. Start adding expenses by tapping the "+" button
  5. Enable expense nudge reminders to stay on track
‚ùì How does iCloud sync work?

iCloud sync keeps your data synchronized across all your Apple devices:

  • Enable iCloud sync in iOS Settings > [Your Name] > iCloud
  • Make sure UangRumah has iCloud access enabled
  • Your expense data will automatically sync between iPhone and iPad
  • Changes made on one device appear on others within minutes
  • Requires active internet connection for syncing
‚ùì Can I export my expense data?

Yes! UangRumah supports data export in two formats:

  • PDF Reports: Professional formatted reports with expense summaries and daily breakdowns
  • CSV Files: Spreadsheet-compatible format for Excel, Numbers, or Google Sheets

To export: Go to Settings > Export Data > Choose format and date range > Share or save the file.

‚ùì How do budget alerts work?

Budget alerts help you stay within your spending limits:

  • Set monthly and weekly budgets in Settings
  • Configure alert thresholds (50%, 75%, 90%, 100%)
  • Receive notifications when you reach each threshold
  • Alerts are calculated in real-time as you add expenses
  • Make sure notifications are enabled in iOS Settings
‚ùì What are cost centers and how do I use them?

Cost centers help you track which payment method you used for each expense:

  • Create cost centers for bank accounts, credit cards, cash, e-wallets, etc.
  • Assign a cost center when adding each expense
  • View spending breakdown by cost center in the dashboard
  • Perfect for couples managing both joint and separate accounts
‚ùì Can I track recurring expenses like subscriptions?

Yes! UangRumah supports automatic recurring expense tracking:

  • Add an expense and mark it as "Recurring"
  • Choose frequency: Daily, Weekly, Monthly, or Yearly
  • Set start and optional end dates
  • The app will automatically track these expenses going forward
  • Perfect for subscriptions, bills, and regular payments
‚ùì What is Privacy Mode?

Privacy Mode helps protect your financial information in public:

  • Dashboard Blur: Blurs sensitive information on the main screen
  • Amount Censoring: Hides specific amounts with asterisks
  • Toggle on/off quickly from the dashboard
  • Set default privacy mode in Settings
‚ùì Is my financial data secure?

Yes, we take security seriously:

  • All data is stored locally on your device by default
  • No data is sent to external servers (except your personal iCloud if enabled)
  • Data is encrypted during iCloud sync
  • We never share your data with third parties
  • Use device passcode or Face ID/Touch ID for additional security

Read our Privacy Policy for complete details.

‚ùì Does UangRumah work offline?

Yes! UangRumah works fully offline:

  • Add, edit, and delete expenses without internet
  • View all dashboard features and reports
  • All data is stored locally on your device
  • Internet only needed for iCloud sync (if enabled)
‚ùì Is UangRumah free?

Currently, UangRumah version 0.1.0 Beta is completely free to use with all features included. Future versions may introduce optional premium features, but we will always maintain a robust free tier.

Troubleshooting Guide

üîÑ iCloud Sync Not Working

If your data isn't syncing between devices, try these steps:

  1. Verify you're signed in to the same iCloud account on all devices
  2. Check iOS Settings > [Your Name] > iCloud and ensure UangRumah is enabled
  3. Make sure you have an active internet connection
  4. Check available iCloud storage (Settings > [Your Name] > iCloud > Manage Storage)
  5. Force close and reopen the app on both devices
  6. Try toggling iCloud sync off and on again
  7. Restart your device

üîî Notifications Not Appearing

If you're not receiving budget alerts or expense nudges:

  1. Go to iOS Settings > Notifications > UangRumah
  2. Ensure "Allow Notifications" is enabled
  3. Check that alert styles are configured (Lock Screen, Notification Center, Banners)
  4. Verify notification settings within the UangRumah app (Settings > Notifications)
  5. Make sure Do Not Disturb / Focus mode isn't blocking notifications
  6. Check that you've set up budget thresholds or reminder schedules

üìä Dashboard Data Not Updating

If your dashboard shows outdated information:

  1. Pull down on the dashboard to refresh manually
  2. Close and reopen the app completely
  3. Check if expenses were added to the correct date
  4. Verify your budget period settings (monthly vs weekly)
  5. If using iCloud sync, wait a few minutes for synchronization

üíæ Data Not Saving

If your expenses or changes aren't being saved:

  1. Make sure all required fields are filled in when adding expenses
  2. Check available device storage (Settings > General > iPhone Storage)
  3. Ensure the app has necessary permissions in iOS Settings
  4. Update to the latest version of UangRumah from the App Store
  5. Restart the app and try again
  6. If problem persists, export your data and contact support

üì± App Crashes or Freezes

If UangRumah is unstable or crashing:

  1. Force close the app and reopen it
  2. Restart your iPhone or iPad
  3. Update to the latest iOS version (Settings > General > Software Update)
  4. Update UangRumah to the latest version from the App Store
  5. Free up device storage if running low
  6. If crashes persist, export your data and reinstall the app
  7. Report the issue to our support team with crash details

üîç Missing Expenses

If you can't find expenses you previously added:

  1. Use the search function to find specific expenses
  2. Check date filters - expenses may be outside your current view
  3. Look in the expense history or version history
  4. If using iCloud sync, check on another device
  5. Pull to refresh the expense list
  6. Review if expenses were accidentally deleted (check recently deleted if available)

üì§ Export Problems

If you're having issues exporting data:

  1. Ensure you have expenses in the selected date range
  2. Check available device storage for PDF/CSV generation
  3. Try a smaller date range if exporting large amounts of data
  4. Make sure you have apps that can handle PDF or CSV files
  5. Try a different export format (PDF vs CSV)
  6. Update the app to the latest version
